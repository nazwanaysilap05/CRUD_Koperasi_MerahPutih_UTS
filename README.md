# 🏛️Sistem Informasi Koperasi Merah Putih
**Tugas UTS - Matakuliah Keamanan Sistem Informasi**

Aplikasi CRUD (Create, Read, Update, Delete) berbasis Web untuk manajemen data anggota koperasi dengan tema visual Merah Putih.

##  Akses Login Admin
Gunakan akun berikut untuk mencoba sistem:
* **Username:** admin
* **Password:** admin123

## 🚀Fitur Utama
* **Login System:** Keamanan akses menggunakan Session PHP (Halaman Utama tidak bisa diakses tanpa login).
* **Interactive Dashboard:** Grafik statistik anggota menggunakan ChartJS.
* **Data Management:** Fitur Lengkap Tambah, Edit, dan Hapus data anggota.
* **Database:** MySQL dengan file konfigurasi di folder `config/`.

##  Cara Menjalankan
1. Download/Clone repository ini.
2. Pindahkan folder ke dalam `htdocs`.
3. Import file database `koperasi_nasional.sql` di phpMyAdmin.
4. Akses melalui browser di `localhost/koperasi_wawa`.
